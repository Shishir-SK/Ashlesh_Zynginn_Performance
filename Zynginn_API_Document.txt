API Controller Reference
  
Spring Kotlin · Base prefix: /api/v1 · 19 controllers
  

Conventions

Convention
Description
Base API Prefix
/api/v1
Public
No JWT required in the controller.
User
Authenticated user context required.
Perm:<NAME>
JWT required and the controller validates the named permission/action set.
Body
Shows the request DTO or multipart parts accepted by the endpoint.
Response
Most handlers return ResponseEntity<Any>. Exact success/error payloads are produced by the underlying service layer.
